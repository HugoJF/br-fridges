# Geladeiras no Brasil

A searchable, sortable reference table of refrigerators available in the Brazilian market, with approximate prices from April 2026.

**[View the table →](https://hugojf.github.io/br-fridges/)**

## What's included

59 models across 9 brands: Consul, Brastemp, Electrolux, Panasonic, Midea, LG, Samsung, Hisense, and HQ.

Each entry covers:

- **Capacity** (liters)
- **Type** — Duplex, Inverse, Side by Side, French Door, Compacta
- **Dimensions** — height, width, depth (cm)
- **Reversible door** — whether the hinge side can be swapped
- **Inverter compressor** — where confirmed
- **Approximate price** (BRL)

## Usage

Open `index.html` in any browser — no build step, no dependencies.

Filters: brand, type, reversible door, and free-text search. Click any column header to sort.

## Data & accuracy

Prices and specs were gathered from retailer listings and manufacturer pages. They reflect availability at the time of research and may be outdated. Use as a starting point for comparison, not as a purchase guarantee.

## Contributing

Pull requests welcome — especially corrections to specs or prices, and new models. Keep the data format consistent with the existing entries in `index.html`.
